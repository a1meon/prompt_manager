# 更新记录（Changelog）

本项目建议遵循「Keep a Changelog」与语义化版本（SemVer）。

## [Unreleased]

## [1.1.1] - 2026-03-15

### Fixed

- 文案：将「创建组」统一为「创建分组」
- 分组：修复分组标题栏「编辑」按钮点击无反应

## [1.1.0] - 2026-03-15

### Added

- 模板分组：创建分组、组展开/折叠、组拖拽排序
- 模板归类：模板可拖入分组，组内模板支持拖拽排序、移出到未分组
- 新建/编辑模板：支持选择所属分组，并在分组选择器内一键创建分组
- 标题栏「帮助」：检查更新与关于弹窗；标题栏显示当前版本小标

### Changed

- 更新检查：支持从 GitHub Releases 获取最新版本并引导更新（安装版支持下载后安装）
- 批量输入：聚焦高亮改为内描边，避免左侧被裁切

### Fixed

- 开发模式：localStorage 非法数据导致空白页的问题修复

## [1.0.1] - 2026-03-15

### Added

- 启动时自动检查 GitHub Releases 更新，下载完成后可一键安装
- 批量生成：目录常驻顶部，滚动时可随时跳转
- 批量生成：变量顺序支持横向拖拽排序；输入规范改为悬停提示

### Changed

- 生成结果：未填写变量不再显示占位或变量样式标记
- UI：模板列表卡片化展示，选中状态更清晰
- UI：编辑模板页导入提示词说明交互优化（移出自动关闭，带箭头）

## [1.0.0] - 2026-03-15

### Added

- 模板管理：创建、编辑、删除、预览
- 变量识别：`{变量名}` 占位与变量提取
- 单条生成与复制结果
- 批量生成、目录跳转与全量复制
- Markdown 编辑与渲染
- 深色/浅色主题切换

### Changed

- 批量生成目录固定在滚动区域顶部，支持随时跳转
- 未填写变量在生成结果中不再显示占位内容
- 编辑模板页导入提示词说明的交互与样式优化（含箭头）
